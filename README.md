# H2O Wastage Prediction with Analytics

## Overview
A web-based application to monitor and analyze water wastage using an interactive dashboard.

## My Contribution
- Designed frontend UI using HTML, CSS, Bootstrap
- Built admin dashboard pages
- Created responsive layouts and forms

## Technologies Used
- HTML
- CSS
- JavaScript
- Python (Flask)
- Flask-SQLAlchemy
- PyMySQL

## Features
- Admin dashboard UI
- Data management interface
- User and complaint management
- Table visualization
